this library aims to encourage **error-as-value**, and explicitly forbids:  

- **SILENT FAILURE**
- **INVARIANT VIOLATION**
- **UNDEFINED BEHAVIOUR**

it requires **C++23** or later, and is compatible w/ `Clang`, `GCC`, `MSVC`
